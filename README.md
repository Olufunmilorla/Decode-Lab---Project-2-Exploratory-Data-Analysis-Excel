# Project 2: Exploratory Data Analysis (EDA) Using Excel

## Overview

This project was completed as part of the Data Analytics Internship Program. The objective was to perform Exploratory Data Analysis (EDA) on a sales dataset to uncover patterns, trends, and insights that support data-driven decision-making.

Using Microsoft Excel, the dataset was cleaned, analyzed, and visualized through Pivot Tables, Pivot Charts, and statistical techniques. The analysis focused on revenue performance, customer behavior, product demand, referral sources, payment methods, and order status distribution.

---

## Project Objectives

* Understand the structure and quality of the dataset.
* Calculate descriptive statistics.
* Identify trends and patterns in sales performance.
* Detect outliers in transaction values.
* Analyze customer purchasing behavior.
* Create an interactive dashboard for business insights.

---

## Exploratory Data Analysis

### Revenue Trend Analysis

Revenue was analyzed across the available years:

| Year |     Revenue |
| ---- | ----------: |
| 2023 | $552,643.24 |
| 2024 | $480,235.87 |
| 2025 | $231,882.85 |

#### Insight

Revenue declined from 2023 to 2025. However, the lower revenue recorded in 2025 should be interpreted cautiously because the dataset contains only partial-year data.

---

### Order Status Analysis

| Order Status | Count |
| ------------ | ----: |
| Cancelled    |   250 |
| Returned     |   247 |
| Pending      |   237 |
| Shipped      |   235 |
| Delivered    |   231 |

#### Insight

Cancelled orders recorded the highest volume, while Delivered orders had the lowest count. Overall, the distribution remained relatively balanced across all order statuses.

### Referral Source Analysis
